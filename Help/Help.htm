Morse Learner (v2014.2.1)

Morse Learner is a small computer program to assist in learning and experimenting with international morse code.  You can use it to transmit or playback morse messages, practice learning to receive morse characters, practice transmitting morse characters, or practice receiving whole words.  Its emphasis is to make learning morse code easy and fun.

Message Mode

This mode lets you: 
- Transmit morse code, either using a morse key (default is the left control button on the keyboard), or by typing the message in on the keyboard.
- Copy your morse code to the clipboard in morse mail format, which preserves the exact timings (the "fist") of the message you recorded.
- Playback text or morse mail messages from the clipboard.

A morse mail message is simply some text which records the time lengths a morse key was pressed down (+) and released (-) for an entire morse code transmission.  An example morse mail message looks like:

<MorseMail>2005.07.07 09:44:36
+75-225+225-75+75-75+75-75+225-225
+75-75+225-225+225-75+225-225+75-75
+225-75+225-75+75-225+75-75+225-75
+75-75+75-225+75
</MorseMail>

The morse learner program doesnt always fully decipher the morse mail message correctly when playing it back, especially if it is very short, or not well transmitted.

Practice Receive Characters Mode

This mode plays the sounds for a morse character, then waits for a small duration waiting for the correct answer to be typed into the keyboard.  A correct answer reduces the chance of that character being shown again, and eventually, many correct answers will unlock additional learning characters.

Practice Transmit Characters Mode

This mode prompts the user with a morse character, then waits for a small duration waiting for the letter to be transmitted correctly with the morse key.  A correct answer reduces the chance of that character being prompted for again, and eventually, many correct answers will unlock additional learning characters.

Practice Receive Words Mode

A more advanced mode, that should only be done once the user has a basic knowledge of morse.  This practice mode sounds the morse code for an entire word at once.  The user can type in the answer as the morse is playing, or for a short duration afterwards.  With every fully correct answer, the chance of longer words being chosen increases.  Words with characters the user is less familiar with have a higher chance of being played. 

Options

Morse Speed

The speed in which morse code is transmitted is mostly commonly measured in "Words per Minute" (wpm), which is the number of times the word "PARIS" can be transmitted in 60 seconds with standard timing for all the pauses between the characters and words.  Morse Learner distinguishes between character speed; the speed in which individual characters are sent, and overall speed; in which the pause lengths between characters and words are increased over their standard lengths to slow down the overall transmission.

Morse is usually sent using standard pause lengths between characters and words, so no distinction is normally made between the two speeds ("character speed" and "overall" speed are the same).  However, for learning, it is useful to be able to increase the lengths between each character, so you get a learn how the character sounds at its proper or fast transmission but still have a bit more time during the pause to decypher it.  The standard "Farnsworth" speed for learning is 12-18 wpm for characters, and 5 wpm overall.

The speed setting controls how the morse is played and the speed in which morse key input is read in Morse Learner.

When learning morse, you will should set a low overall more speed (5pm), and the character speed to the final speed you want to eventually become familiar with.  As your learning progresses you will slowly increase the overall speed to the character speed.

Difficulty

Controls the length the program waits for a correct answer in the practice mdoes.

Sound

This section controls the sound output frequency which is used play the morse code. 

Input

What input device and settings to use as for the morse key input.

Learning Characters

All the practice modes will pick characters only from the "learning character" set.  As you provide correct answers in the practice modes, the bar for those characters in the display will grow smaller.  The levels can be reset to the predefined amounts; "beginner knowledge", "intermediate knowledge", or "advanced knowledge", for the currently selected character set through the menu.

The practice receive characters mode and practice send characters mode will automatically add new characters to this set once all the existing selected set has been learnt roughly to an "advanced" level.

Different characters can also be manually selected/unselected with the mouse, or groups of characters (alphabetic, numberic, punctuation) can be added or removed through the menu. 

Mouse & Keyboard Shortcuts

Esc - Quit the current mode or message playback and clears the current message.
Right Control - The default keyboard morse key (works even if the program is running in the background).
F1 - Message Mode.
F2 - Practice Receive Characters Mode.
F3 - Practice Transmit Characters Mode.
F4 - Practice Receive Words Mode.
F10 - Options.
Control+C - Copy the current message while in message mode to the clipboard in morse mail format.
Control+X - Copy the current message while in message mode to the clipboard in text format.
Control+V - Play the text or morse mail message currently on the clipboard.
Control+C x2 - While in another application, plays whatever is copied onto the clipboard if Morse Learner is running in the background.
Space - Pause/Restart the Practice Mode.

Clicking the mouse of individual characters in the "learning characters" section will lock/unlock them individually.

Connecting a Morse Key to your computer

Although Morse Learner lets the user input morse code using either the keyboard or a mouse button, nothing beats the feel and authenticity of a real morse key.  A morse key or paddle can be connected through the serial or COM port of your computer on either the "DSR" or "CTS" lines.

9 pin serial cables can be brought from an electronics store, or often obtained cheaply from old second-hand dial-up modems.  To detemine which pair of wires to connect is sometimes a matter of experimentation, since the color coding of the wires is not always consistent.  One simple method is to strip all the wires, and try touching different wires together until the Morse Learner program confirms a contact on the right pair (make sure you turn on the com port input in the options first).  Once the correct wires are confirmed, connect the wires to the morse key, and cut the rest short.

Iambic morse paddles are not (yet) supported.
